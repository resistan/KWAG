웹 사이트의 접근성을 효과적으로 향상시킬 수 있는 웹 접근성 전문가 평가를 진행합니다. 접근성 평가는 물론이며, 접근성 평가 방법론이나 평가 프로세스 및 자동 평가 도구 개선에 대한 내용 전반을 다룹니다.
웹 접근성 평가는 웹 사이트 신규 제작이나 개편시 반드시 필요한 과정 중 하나입니다. 특히 제작 초기부터 제작 과정 내내 진행하는 웹 접근성 평가는 제작 완료 후에 진행하는 것에 비해 웹 사이트의 접근성을 향상시키기 위해 보다 효율적입니다.
웹 접근성 평가 TF는 특성상 외부에 비공개로 별도의 평가 TF 그룹에서 활동합니다. 웹 접근성 평가 나 평가 TF에 관심이 있거나 문의 사항이 있으면 아래 이메일이나 사이트에 연락할 수 있습니다.
