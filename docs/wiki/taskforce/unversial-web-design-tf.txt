목적

웹 디자인의 철학으로써 Universal Web Design을 정립하는 것이 본 TF의 목적이다. 이를 위해서 도시환경, 건축 내·외부, 가구, 제품, 보조기구 등의 디자인에서 중요 철학으로 인식되고 있는 Universal Design의 개념을 웹에 맞게 재정의한다. 그 첫번째 결과물로 Universal Web Design 선언문을 작성하여, 웹에 개시하고 이를 국내외 웹 디자이너에게 알린다. 그 후 Universal Web Design 가이드를 작성하고 이를 웹 사이트에 개시한다. 또한 Creative Commons의 선진적인 활동에 이어가도록 노력한다.
