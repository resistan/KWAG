내가 속한 조직의 사이트 또는 여러 사람들에게 유용한 콘텐츠나 서비스를 제공하는 웹 사이트를 중심으로 웹접근성 개선이 필요한 곳을 선정하여 대안을 문서화. 그리고 웹접근성 분석과 개선안을 정리하는 것에서 멈추는 것이 아닌 해당 웹 사이트에 직접 의견을 전달하고 사이트 개선을 목표로 합니다.
KWAG Task Force (기존)

웹 접근성 평가 TF

웹 사이트의 접근성에 대한 전문가 평가를 진행합니다. 실질적인 접근성 평가는 물론이며, 접근성 평가 방법론 및 평가 프로세스 전반에 대한 논의도 함께 합니다. 웹 접근성 평가는 웹 사이트 신규 제작이나 개편시 반드시 필요한 과정 중 하나입니다. 특히 제작 초기부터 제작 과정 내내 진행하는 웹 접근성 평가는 제작 완료 후에 진행하는 것에 비해 웹 사이트의 접근성을 향상시키기 위해 보다 효율적입니다.
Universal Web Design TF

웹 디자인의 철학으로써 Universal Web Design을 정립하는 것이 본 TF의 목적이다. 이를 위해서 도시환경, 건축 내·외부, 가구, 제품, 보조기구 등의 디자인에서 중요 철학으로 인식되고 있는 Universal Design의 개념을 웹에 맞게 재정의한다. 그 첫번째 결과물로 Universal Web Design 선언문을 작성하여, 웹에 개시하고 이를 국내외 웹 디자이너에게 알린다. 그 후 Universal Web Design 가이드를 작성하고 이를 웹 사이트에 개시한다. 또한 Creative Commons의 선진적인 활동에 이어가도록 노력한다.
웹 접근성 지침 2.0 개발 TF

W3C, 미국 재활법 508조 등 웹 접근성 관련 국제 지침들이 2.0 버전을 개발하고 있는 중입니다. 우리나라에서도 "인터넷 웹 콘텐츠 접근성 지침 1.0"을 2005년에 만들었습니다만, 현재 우리나라의 보조기술, 브라우저의 기술 변화, 새로운 신기술 등을 반영하지 못하고 있는 실정입니다. 본 TF에서는 웹 접근성 관련 국내외 지침(W3C, 미 재활법 508조 기준, JIS X 8341-3 등)을 분석하고, 이를 기반으로 2007년 세계가 부러워하는 웹 접근성 지침을 개발합니다.
게임 웹사이트의 접근성향상을 위한 TF

게임사이트는 그 필요성에 의해 접근성은 아직 고려되지 않고 제작되어 왔습니다. 하지만 게임을 즐기기위한 정보를 얻는 입장에서의 사이트의 목적을 봤을때 접근성은 필요한 요건이라고 생각됩니다. 지금까지 상대적으로 소홀히 되었던 정보 격차 해소에 대한 해법은 무엇인지 논의하고 해결점을찾아서 가이드를 제작하는 것이 TF의 목적 입니다.
바른 누리틀 만들기 TF

웹 에디터 접근성 향상을 위한 태스크 포스 팀입니다. 웹에서 저작도구로 이용되는 웹 에디터의 올바른 모델과 기존 에디터들이 개선해야 할 점 등을 제시할 수 있도록 웹 에디터를 여러 각도를 통해 연구하고 가이드라인을 설정하는 것을 목적으로 합니다.
JavaScript Accessibility Guideline Interface Library

자바 스크립트의 접근성 향상을 위한 가이드 라인 제시 및 인터페이스 라이브러리 구현을 목적으로 결성되었습니다. TFT명은 JAGIL을 그대로 읽으시며 "재~길" 입니다.
영상 자막 접근성 TF

영상 자막 접근성 TF는 점점 널리 쓰이고 있는 동영상과 오디오의 접근성을 향상시키기 위한 자막, 캡션, 음성 설명에 대한 연구 모임입니다. 단기적으로는 멀티 미디어 재생기들의 접근성 기능 조사, 웹에서 접근성을 고려한 멀티미디어 포함 방법, 자막 제작 프로그램 현황 조사 및 사용법 정리, 인간의 지각적/인지적 특성을 고려한 캡션의 효과적인 제시 방법 등을 다루려고 하고 있으며, 향후에는 멀티미디어 요소를 제어하는 SMIL의 응용, 동영상 검색과 색인을 향상시키기 위한 동영상 메타데이터 및 캡션 제공 방법, 이러닝 등 응용 분야 멀티미디어 자료의 접근 가능한 제시 방법 등에 대해 함께 공부할 계획입니다.
웹 디자인 TF (Universal Web Design TF로 활동 이전)

웹 사이트의 접근성을 증진시키는 디자인과 관련된 활동을 진행합니다. 웹 접근성과 폰트 같은 타이포나 이미지, 색상, 레이아웃 등 웹 디자인 요소들 사이의 관계를 정리하고, 접근성을 증진시킬 수 있는 웹 디자인과 관련된 내용을 정리합니다.
